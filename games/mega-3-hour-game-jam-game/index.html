Mega 3 Hour Game Jam Game

3 Hour game jam run by MEGA , all participants had 3 hours to make a 4 player local multiplayer game. At the end of the jam, all the games were compiled into a single WarioWare/Mario Party style mix. Bethany Martin and I made the game Horny Chix. Make as many offspring with your opponents before time runs out. Negative points for mating with offspring.
